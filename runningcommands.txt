## üîß Alternative Commands

### Development Mode (with auto-reload)
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload --log-level debug
```

### Production Mode (no reload)
```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

### Custom Port
```bash
uvicorn main:app --host 0.0.0.0 --port 3000 --reload
```

### Bind to Specific IP Address (192.168.110.173)
```bash
uvicorn main:app --host 192.168.110.173 --port 8000 --reload
```

**Note:** Using `--host 0.0.0.0` (recommended) allows access from all network interfaces including 192.168.110.173. Using `--host 192.168.110.173` binds only to that specific IP address.

## üåê Server URLs

- **Local:** http://localhost:8000
- **Network:** http://192.168.110.173:8000
- **API Docs:** http://localhost:8000/docs or http://192.168.110.173:8000/docs
- **Health Check:** http://localhost:8000/health or http://192.168.110.173:8000/health
